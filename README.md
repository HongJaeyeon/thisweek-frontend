# 해커톤 개발 주제 : 청춘 !

데이트를 할 때마다 뭐할지 고민 중인 청춘들을 위해 장르별로 이번 주의 문화 활동을 알려준다

# front-end

### 사용 기술

- react.js
- react-router-dom
- react-hook-form
- tailwind CSS

### 구현 기능

- 카테고리별 문화 활동 랜더링
- 회원가입, 로그인 기능
- 포스트잇 형식의 글쓰기 기능
